# AQI Prediction Using ANFIS | Cloud Deployment on AWS EC2

An intelligent **Air Quality Index (AQI) Prediction System** built using **ANFIS (Adaptive Neuro-Fuzzy Inference System)** and deployed on **AWS EC2** using **Streamlit**.  
This project combines **Soft Computing**, **Machine Learning**, and **Cloud Computing** to provide real-time AQI prediction with health advisories through a live web application. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1}

---

## 🌍 Project Overview

Air pollution is a growing concern in urban areas. Monitoring and predicting AQI can help individuals and governments take preventive action.

This project uses a custom-trained **ANFIS model** to predict AQI from environmental pollutants:

- PM2.5
- PM10
- NO₂
- SO₂
- CO
- O₃

The model is trained on real-world Indian air quality datasets and deployed as a web app accessible from any browser.

## 🎯 Key Objectives

- Build an intelligent AQI prediction system using ANFIS
- Learn fuzzy logic + neural network hybrid modeling
- Train on real Indian AQI dataset
- Develop an interactive web app using Streamlit
- Deploy the app on AWS EC2
- Demonstrate end-to-end ML + Cloud workflow

---

## 🧠 What is ANFIS?

**ANFIS (Adaptive Neuro-Fuzzy Inference System)** is a hybrid model that combines:

- **Fuzzy Logic** → Human-like rule-based reasoning
- **Neural Networks** → Learning from data

It provides:

✅ High Accuracy  
✅ Interpretability  
✅ Fast Training  
✅ Better handling of nonlinear relationships
